# news-aggregator 
##  Project Overview
The MERN News Aggregator is a web-based application that collects and displays real-time news articles from different categories such as Technology, Business, Sports, Health, and Entertainment.  
The project uses the MERN stack and integrates an external News API to provide up-to-date news in a clean and user-friendly interface.

This project was developed as part of the CodeClause Web Development Internship 

---

##  Features
- Category-based news filtering
- Real-time news fetched from News API
- Responsive and clean UI
- External “Read More” links to original articles
- Backend API built using Node.js and Express
- Secure handling of API keys using environment variables
